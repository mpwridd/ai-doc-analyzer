# AI Document Analyzer 📄

Upload, analyze, and understand any document with **Mimo v2.5 Pro** AI.

## Features

- 📝 **Summary** — Executive summaries with key points
- 💡 **Insights** — Deep analysis, patterns & implications
- 🎭 **Sentiment** — Tone, emotion, bias detection
- 🏷️ **Keywords** — Topic & theme extraction
- ❓ **Questions** — Auto-generate comprehension Q&A
- 💬 **Q&A Chat** — Ask anything about the document
- 🌐 **Translate** — Multi-language translation
- 📁 **File Upload** — .txt, .md, .csv, .json, .xml, .html, .log
- 💾 **Export** — Download analysis results
- 📜 **History** — Previous analyses tracking

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript** + **Tailwind CSS**
- **Lucide React** (icons)
- **Mimo v2.5 Pro** (AI model)

## Getting Started

```bash
npm install
npm run dev
```
